# **App Name**: Avatar Snake

## Core Features:

- Avatar Snake Display: Display a snake avatar that grows when "eating" a taco emoji.
- Snake Control: Control the snake's movement using arrow keys or touch gestures.
- Taco Display: Display the taco emoji as food for the snake to consume.
- Game Over Logic: Implement game over conditions (e.g., snake hitting the wall or itself).
- Score Display: Display the current score during the game.
- Random Taco Placement: Randomly place a taco on the grid.

## Style Guidelines:

- Primary color: Use a deep forest green (#228B22) to create a sense of lush environment and depth.
- Background color: A desaturated, dark grayish-green (#2B362B) to complement the snake, derived from the hue of the primary color.
- Accent color: Use a warm, analogous yellow-green (#9ACD32) to highlight interactive elements, contrasting with the dark background and drawing user attention.
- Display a clean and readable sans-serif font for the score.
- A grid-based layout for the game board to facilitate the snake's movement.
- Smooth transitions and animations for snake movement and taco appearance.
- A distinct avatar (user profile image) used for the snake head and body segments, repeating to form the snake.
